Sobre

Apresento o Unitasks, um sistema intuitivo de gerenciamento de tarefas, concebido como projeto universitário para aprimorar habilidades em HTML, CSS e JavaScript. A plataforma permite que usuários efetuem login, adicionem novas tarefas e consultem sua lista de pendências de maneira eficiente. O projeto explora ainda a organização financeira e um acervo de conteúdos, aplicando tecnologias como PHP e banco de dados para uma experiência web completa.
